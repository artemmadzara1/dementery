# dementery
проект для выявления признаков похожих на деменцию(не диагноз надо проходить обследование у врача)
срок проекта прибризительно до конца апреля 2026 года
Календарный план

На первой неделе настраивается сервер и создается базовая страница проекта. 

На второй неделе добавляются первые вопросы и простая проверка ответов. 

На третьей неделе реализуется выбор между бесплатной и полной версией теста. 

На четвертой неделе внедряются разные форматы вопросов, включая выбор варианта и ввод даты. 

На пятой неделе добавляется подсчет результатов по отдельным когнитивным навыкам. 

На шестой неделе внедряется текстовая расшифровка итогов в зависимости от баллов. 

На седьмой неделе добавляется отчет о прохождении. 

На восьмой неделе загружаются все двадцать пять вопросов и завершается оформление интерфейса.



модули
Данные вопросов Хранение вопросов для тарифов Free и Pro.

Состояние Переменные сессии: тариф, ответы, индекс, время.

Утилиты Безопасность HTML и нормализация текста.

Тарифы Переключение версий и сброс теста.

Интерфейс Отрисовка типов вопросов (текст, дата, слайдер).

Навигация Переходы назад/вперёд, валидация заполнения.

Ввод Сбор ответов пользователя в массив.

Скоринг Проверка ответов, подсчёт баллов и доменов.

Результаты Показ счёта, графиков, советов.

Инициализация Запуск приложения при загрузке страницы.
